PHP-HTML::Template

[docs]|[samples]|[project]

PHP-HTML::Template is the implementation of a template system for use within PHP scripts, where syntax of templates is the same as that of Sam Tregar's HTML::Template Perl module.

PHP-HTML::Template is an Open Source project hosted on SourceForge.net. The aim of this project is to create a PHP extension module.

In its current state, PHP-HTML::Template is a set of classes. You can see some examples in our samples page. You will want to visit our documentation page as well.

You are invited to participate in this project. You can help in any of the following ways:

  • Download the latest release of the library, test it and report the bugs you find.
  • Help us improve the source code.
  • Help us create the PHP extension module in C. This is the main objective of the project.
  • Participate in the mailing list or the public forums.
